# TaskPilot

A production-ready AI agent system built with FastAPI, LangChain, and Docker. Features a supervisor pattern that intelligently routes tasks between specialized agents for email management and research.

## Features

- **Supervisor Agent**: AI-powered routing between specialized agents
- **Email Agent**: Automated email sending and inbox management
- **Research Agent**: Information gathering and analysis
- **FastAPI Backend**: RESTful API with database persistence
- **Docker Compose**: Multi-service containerized deployment
- **PostgreSQL**: Persistent data storage with SQLModel ORM
